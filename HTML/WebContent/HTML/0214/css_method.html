class를 이용한 스타일 설정


    style에 class를 선언한다.


    addClass(class이름)     : class이름을 추가
    removeClass(class이름)  : class이름을 제거
    toggleClass(class이름)  : class이름을 추가, 삭제를 번갈아 한다.
    hasClass(class이름)     : 해당 class이름이 있는지 여부 확인
