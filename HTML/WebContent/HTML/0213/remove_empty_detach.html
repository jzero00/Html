비우기와 지우기

	
	empty()	: 대상 요소는 삭제하지 않고 자식들만 지운다.
	remove(): 대상 요소뿐만 아니라 자식 요소까지 함께 지워진다.
오늘은 목요일 너무 졸립다.. 집가고 싶다...

삭제시 이벤트 확인

		
	remove()	: 요소 삭제시 이벤트 핸들러로 삭제
	detach()	: 요소 삭제시 이벤트 핸들러는 삭제되지 않는다

멍하니 아무 일도 할 일이 없어 이게 이별인 거니

전화기 가득 찬 너와의 메시지만 한참 읽다 읽다

너의 목소리 마치 들린 것 같아 주위를 둘러보면

내 방엔 온통 너와의 추억투성이 이제야 눈물이

수고했어 사랑 고생했지 나의 사랑 우리 이별을 고민했던 밤

서로를 위한 이별이라고 사랑했단 너의 말을 믿을게

혹시 너무 궁금해 혹시 너무 그리우면 꼭 한 번만 보기로 해

복사시 이벤트 확인

		
	clone()		: 요소 복사할때 이벤트 핸들러는 복사되지 않는다.
	clone(true)	: 요소 삭제할때 이벤트 핸들러도 함께 복사된다.
