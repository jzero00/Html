배열 합치기 / 배열을 문자열로 변환하기 
    a.concat(b) : a배열과 b배열을 합쳐서 새로운 배열을 만든다.
    a.join(b)   : a배열과 b배열이 하나의 문자열로 변환
    a.join('/') : aaa/aaa/aaaa/aaaa/aaaa/aaa/aa/aaaa
