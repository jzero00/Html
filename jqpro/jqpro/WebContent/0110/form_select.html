tag이름이 select요소로 접근, 값을 가져온다.
    document.getElementsByTagName('select')[0].value

    a = document.getElementsByTagName('select')[0]
    x = a[0].value

name이 namelist요소로 접근, 값을 가져온다.
    document.getElementsByName('namelist')[0].value

    a = document.getElementsByName('namelist')
    x = a[0].value

tag이름이 div인 요소로 접근, 출력한다.
    b = document.getElementsByTagName('div')
    b[0].innerHTML = x;
